Tailored Features for Modern Displays

A fully featured monitor management utility that works quietly in the background without getting in your way.

Per-Monitor Control

Configure brightness offsets (0–90%) and toggles for each screen independently. Drag a slider to automatically enable dimming on that display.

Intelligent Idle Fade

Fades screen overlays smoothly when no user inputs are detected. Prevents static burn-in by dimming down and hides your mouse cursor automatically.

Smart Video Bypass

Automatically detects video playback or audio sessions from major media players and browsers (Chrome, Edge, Firefox). Bypasses dimming on active screens.

Fullscreen & Game Mode

Suspends dimming on monitors running fullscreen games or presentations. Zero interference during gameplay.

Automated Scheduling

Schedule dimming times automatically using local 15-minute interval sliders. Ideal for overnight shifts or late-night study sessions.

Zero Overhead & Privacy

No telemetry, no tracking, and no internet access needed. Runs as a lightweight native binary (~150 KB, ~3 MB memory footprint).

Specifications

Built with efficiency and reliability in mind. Zero bloat.

Executable Size ~150 KB (standalone), ~140 KB (MSIX package)
Memory Footprint ~3 MB working set at idle
Dependencies None. Statically linked native Win32 binary.
Supported Systems Windows 10 (1809+) and Windows 11 (x64)
Hardware Footprint 0% CPU load at idle (throttled monitoring ticks)
Installation Scope Per-user local sandbox or Program Files (no Admin UAC required)
