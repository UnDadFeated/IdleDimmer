v1.3.9 — Seven bug fixes, working per-monitor control, active dimming restored.

• groupDim default changed to false — each monitor toggles independently
• Ctrl+Alt+D hotkey now calls SetDimmingEnabled() so it actually works
• Idle timer no longer skips when video is detected (idle state doesn't stick)
• Idle state cleared when idle dim is disabled (was stuck forever)
• Overlay idle branch checks info->enabled (disabled monitors stay off during idle)
• Overlay idle branch uses max(dimValue, idleDimLevel) (per-monitor slider respected)
• Update check thread data race fixed (PostMessage instead of direct write)
